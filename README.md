## Installation
## Requirements
* nodejs 8.x.x
* mongodb
`$ npm install`
### Running the app
```
  # development
  $ npm run start

  # watch mode
  $ npm run start:dev

  # production mode
  $ npm run start:prod
```
### Running the app
```
  # unit tests
  $ npm run test

  # e2e tests
  $ npm run test:e2e

  # test coverage
  $ npm run test:cov
```
